# PhoneTool
遍历后n位号码

```
$ python3 phone_tool.py -h                          

        手机尾号遍历小工具V1.0

        -p      前数位固定数字，例：-p 1300610 或 --phone=1300610
        -i      输入号段文件，例：-i read.txt 或 --input=read.txt
        -o      输出文件，例：-o wuhan.txt 或 --output=wuhan.txt
        -t      写入线程，默认10线程，例：-t 12 或 --thread=12
        
```
